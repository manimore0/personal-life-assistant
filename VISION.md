# Vision

Build a personal life assistant that acts as a central system to manage and improve daily life.

## Core Idea
Instead of relying on multiple disconnected tools, create a single system that:
- Tracks tasks and responsibilities
- Provides structure to daily and weekly planning
- Helps improve habits and decision-making

## Long-Term Goals
- Unified life dashboard
- Intelligent planning assistance
- Personalized recommendations
- Automation of repetitive decisions

## Guiding Principles
- Build for personal use first
- Keep things simple and usable
- Avoid unnecessary complexity
- Use AI selectively, not everywhere
- Focus on consistency over perfection

## What This Is NOT
- Not a full-scale SaaS product (for now)
- Not an overengineered AI system
- Not dependent on heavy API usage
